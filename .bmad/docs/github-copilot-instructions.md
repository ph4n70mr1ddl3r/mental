# BMAD Method - GitHub Copilot Instructions

## Activating Agents

BMAD agents are installed as prompt files in `.github/agents/`.

### How to Use

1. **Open Chat View**: Click Copilot icon in VS Code sidebar
2. **Select Prompt File**: Use `@` mention or prompt file selector
3. **Choose Agent**: Reference the agent file from `.github/agents/`
4. **Chat**: Agent is now active for this session

- `bmd-custom-bmm-architect.agent.md` - System Architect
- `bmd-custom-bmm-dev.agent.md` - Developer
- `bmd-custom-bmm-pm.agent.md` - Product Manager
- `bmd-custom-bmm-sm.agent.md` - Scrum Master
- `bmd-custom-bmm-tea.agent.md` - Test Architect
- `bmd-custom-bmm-tech-writer.agent.md` - Technical Writer
- `bmd-custom-bmm-ux-designer.agent.md` - UX Designer
- `bmd-custom-bmm-quick-flow-solo-dev.agent.md` - Quick Flow Solo Dev
- `bmd-custom-core-bmad-master.agent.md` - BMad Master

### Notes

- Agents persist for the chat session
- Switch agents anytime by referencing a different prompt file
- Multiple specialized agents available
